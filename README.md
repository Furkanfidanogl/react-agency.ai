# 🚀 Agency.ai Landing Page

> A modern, minimalist, and responsive single-page landing page designed for digital agencies. Built as a practical exercise to implement component-based UI and master utility-first CSS workflows.

---

## 🎯 Key Focus Areas

*   🧩 **Modular UI:** Engineered with a component-based architecture ensuring high maintainability and clean code organization.
*   🎨 **Utility-First Styling:** Styled extensively using **TailwindCSS** for rapid, responsive design implementation without relying on custom CSS files.
*   ⚡ **Modern Build System:** Powered by **Vite** to provide lightning-fast Hot Module Replacement (HMR) and highly optimized production builds.

---

## 🛠️ Technologies Used

| Technology | Role | Description |
| :--- | :--- | :--- |
| ⚛️ **React** | Framework | Frontend architecture for building interactive user interfaces. |
| ⚡ **Vite** | Build Tool | Next-generation frontend tooling for rapid development and bundling. |
| 🌬️ **TailwindCSS** | Styling | Utility-first CSS framework for efficient and responsive styling. |
| 🧹 **ESLint** | Linter | Pluggable linting utility to enforce code quality and consistency. |

## 💻 Local Setup & Installation

Follow these steps to get the project running on your local machine:
```bash
# 1. Clone the repository
git clone [https://github.com/Furkanfidanogl/react-agency.ai.git](https://github.com/Furkanfidanogl/react-agency.ai.git)

# 2. Navigate to the project directory
cd react-agency.ai

# 3. Install necessary dependencies
npm install

# 4. Start the development server
npm run dev
